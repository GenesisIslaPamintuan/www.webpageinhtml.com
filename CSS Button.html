<title>CSS Button Animation Example</title>
</head>

CSS Code

<style>
.animate-btn {
padding: 10px 20px;
font-size: 16px;
background-color: skyblue;
border: none;
border-radius: 5px;
cursor: pointer;
transition: background-color 0.3s ease-in-out;
animation: scaleButton 2s ease-in-out infinite;
}

.animate-btn:hover {
background-color: dodgerblue;
}

@keyframes scaleButton {
0% {
transform: scale(1);
}
50% {
transform: scale(1.2);
}
100% {
transform: scale(1);
}
}
</style>
